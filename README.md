Automated Web Server Deployment using Bash
Project Overview
This project demonstrates how to automatically deploy a web server on a linux system using a Bash script. The script installs and configure a web server, starts the service, and verifies that the server is running correctly.This project was created as part of my Linux system Administration partice while prepearing for RHCSA level administration tasks 

Architecture
The architecture of this project is simple and focuses on a single Linux server environment.

1. The Bash script is executed on a Linux machine.
2. The script installs the required web server packages.
3. The web server service is started and enabled.
4. The server status is verified to confirm successful deployment.
Tools Used
• Linux Operating System
• Bash Scripting
• Package Manager (dnf / apt)
• Systemctl (Service Management)
